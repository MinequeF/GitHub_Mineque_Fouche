CALCULATOR - README

WHAT IS THIS?
A simple calculator that runs in command line. Add, subtract, multiply, or divide numbers.

HOW TO RUN IT

Windows:
1. Open Command Prompt
2. Go to the folder with calculator.py
3. Type: python calculator.py

Mac/Linux:
1. Open Terminal
2. Go to the folder with calculator.py
3. Type: python3 calculator.py

HOW TO USE IT

1. Enter first number
2. Enter second number
3. Choose: + (add), - (subtract), * (multiply), / (divide)
4. See result
5. Type y to do another calculation, or n to quit

EXAMPLE

Enter first number: 10
Enter second number: 5
Choose operation (+, -, *, /): +

Result: 10.0 + 5.0 = 15.0

Do another calculation? (y/n): n

Thanks for using the calculator. Goodbye!
